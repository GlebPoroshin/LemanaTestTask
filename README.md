# Тестовое задание Lemana

Это репозиторий **тестового задания в Lemana**! Это приложение для интернет-магазина с такими функциями, как отображение списка товаров, управление списком покупок и корзиной, при этом соблюдены принципы чистой архитектуры и паттерн проектирования MVI.

## Стек технологий

### Основные технологии
- **Kotlin Multiplatform**
- **Compose Multiplatform**
- **SQLDelight**
  
### Взаимодействие с сетью
- **Ktor**
  
### DI
- **Koin**

### Навигация
- **Voyager**

### Утилиты
- **Napier**

### Ключевые классы

- **MainScreenModel**: Управляет состоянием для экрана со списком товаров.
- **ShoppingListRepository**: Управляет данными для списка покупок.
- **CartRepository**: Управляет данными для корзины.
